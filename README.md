# admin-spa-react

react spa manager cli

## Base on

* [create-react-app](https://github.com/facebook/create-react-app)
* [rematch](https://github.com/rematch/rematch)
* [antd](https://ant.design/)

## Features

* 登录（同步、异步）
* 菜单路由（配置、权限）
* 数据流
* 权限
* 常用增删改查

## Build

* 可直接打包静态资源、发布cdn
* 可启express服务

## Dev

* 异步跨域调用
* 启用express代理转发
